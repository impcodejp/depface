# プロジェクト概要
AxumベースのRest APIサーバ

# 技術スタック
- 言語: Rust (edition 2021)
- Webフレームワーク: Axum
- 非同期ランタイム: Tokio
- シリアライズ: serde / serde_json
- エラー処理: thiserror / anyhow
# 以下は使っていれば追記
# - DB: sqlx + PostgreSQL
# - 認証: JWT (jsonwebtoken)

# よく使うコマンド
- 起動: cargo run
- テスト: cargo test
- Lint: cargo clippy
- フォーマット: cargo fmt
- ビルド確認: cargo check
